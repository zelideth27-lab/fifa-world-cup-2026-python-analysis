# FIFA World Cup 2026 Python Analysis ⚽📊

This repository contains the Jupyter Notebook used to analyze FIFA World Cup 2026 statistics with Python.

## Technologies

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Notebook

- 📓 01_World_Cup_2026_General_Statistics.ipynb

The notebook includes:

- General tournament statistics
- Goals per match
- Yellow and red cards
- Offensive efficiency analysis
- Team offensive profile visualization

## Dataset

This project uses the FIFA World Cup 2026 Dataset available on Kaggle:

https://www.kaggle.com/datasets/jodach/fifa-world-cup-2026-dataset
